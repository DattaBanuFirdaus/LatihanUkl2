Program di atas berfungsi untuk menentukan apakah suatu bilangan termasuk bilangan genap atau ganjil.
Cara kerjanya yaitu pengguna memasukkan sebuah bilangan bulat melalui keyboard.
Program kemudian memeriksa sisa hasil bagi bilangan tersebut dengan 2 menggunakan operator %.
Jika hasilnya 0, berarti bilangan tersebut genap; jika tidak, berarti ganjil. Setelah itu, program menampilkan hasilnya di layar
